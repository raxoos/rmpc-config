# rmpc-config
My personal rmpc configuration with a OneDark theme.

## Layout & Tweaks
* Clean, full-width layout without the default splits.
* Custom header with ratings, playcount, and track info.
* Configured with shortcuts for fast navigation (Queue, Artists, Albums, etc.).
* Uses Nerd Font icons.

## File Structure

The files should be placed in your `~/.config/rmpc/` directory like this:

```text
~/.config
└── rmpc
    ├── config.ron
    └── themes
        └── onedark.ron
